# Audio Player

This project is made using MERN (MongoDB, Express, Reactjs, Nodejs) with Tailwind.

### To clone this project locally

```bash
git clone https://github.com/sethshivam11/audio-player
```

### To run this project
```bash
npm run build
npm start
```

### Environment variables required to run this
- `MONGODB_URI` - Your MongoDB URL
- `PORT` - Any port
- `NODE_ENV` - Node environment (expected value 'production')
